web table
a HTML structure for creating rows and columns on a Web page
table row
a horizontal group of values within a table
table column
a set of data values of a particular simple type, one value for each row of the database
table definition
the td element defines a table cell
rowspan
specifies the number of rows a cell should span vertically
colspan
the number of columns a cell should span (or merge) horizontally
rowgroup
the header cell provides header information for the rest of the row group that contains it.
Colgroup
the header cell provides header information for the rest of the column group that contains it.
